Cellosoft JTablet Plugin Java Extension for Tablets

For Artists

For Java Developers

JTablet provides a simple open-source (zlib license) interface for accessing tablet input in your Java applications.

JTablet 2.0 SDK (alpha API release)

JTablet 2 provides new event-based API using JPen to provide cross-platform driver support.

The JTablet 2.0 API is still being finalized, so please e-mail me any comments or suggestions on the API!

New features of JTablet 2

  • Event-based API
  • Fractional cursor coordinates
  • Full input rate (200 points per second on some tablets)
  • Compatibility layer for JTablet 0.9.5 (supporting legacy applets on Mac OS X)
  • Crossplatform: Windows, Mac OS X, and Linux support

JTablet 0.9.5 SDK [deprecated]

JTablet 0.9.5 provides a polling API for Windows (WinTab). Jason Gerecke provides a Linux version.

This version has been deprecated. It will no longer be supported once JTablet 2.0 launches.
