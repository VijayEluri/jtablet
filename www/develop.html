Cellosoft JTablet Java Extension for Tablets

For Artists

For Java Developers

JTablet provides a simple open-source (zlib license) interface for accessing tablet input in your Java applications.

JTablet 2 SDK (alpha API release)

JTablet 2 provides new event-based API using JPen's native code to provide cross-platform support.

JTablet 2 is currently in development and the API should be considered experimental (it may change!). Please e-mail me any comments or suggestions on the API.

New features of JTablet 2:

  • Event-based API
  • Fractional cursor coordinates
  • Full tablet input rate (as much as 200 points per second on some tablets)
  • Compatibility layer for JTablet 0.9.5 (allowing legacy applets to now be supported on Mac OS X)
  • Crossplatform: Windows, Mac OS X, and Linux support

JTablet 0.9.5 SDK [deprecated]

JTablet 0.9.5 provides a polling-based API aimed at WinTab on Windows with a Linux port by Jason Gerecke.

With the advent of JTablet 2.0, this version has been deprecated and will no longer be supported come launch of JTablet 2.0.
